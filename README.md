### Swiss Ephemeris
Swiss Ephemeris calculator compiled for Ubuntu in folder:
**panch

### Configuration
intermediate calculations stores in MySQL
Configuration file for DB and etc:
**noindex/configuration.php

### installation
all Holydays explanation stored in DB
